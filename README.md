# ShadowSocks上屏蔽广告
屏蔽广告+统计服务+视频广告+APP广告 使用 `hosts` 这个文件
这个文件主要包含：
* `app.hosts` 
* `trace.hosts`
* `video.hosts`
* `web.hosts`
* https://github.com/vokins/yhosts 中的`hosts`

`adservers.hosts` 是国外的广告服务
`mvps.hosts` 是国外的恶意软件
